
Food Delivery Web Application

This is a simple Food Delivery web application built using Python and Django.  
Users can browse restaurants, view menus, add items to the cart, and place orders.  
Admins can manage restaurants, menus, and track customer orders.

Features:
- User registration and login
- Restaurant listing and menu display
- Add to cart and order placement
- Admin panel to manage restaurants and menu items
- Order tracking system

Technologies Used:
- Python
- Django
- SQLite Database
- HTML, CSS, JavaScript
- Bootstrap

How to Run the Project:
1. Clone the repository
2. Create a virtual environment
3. Install requirements:
   pip install -r requirements.txt
4. Run migrations:
   python manage.py migrate
5. Start the server:
   python manage.py runserver
